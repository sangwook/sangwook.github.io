Facebook의 새로운 뉴스피드 알고리즘 "Story Bumping"

2013-08-07 이상욱

문제

  • 최근까지 뉴스 피드는 사용자가 이전에 로그인 한 이후의 모든 게시물을 평가하여
  • 가장 관심이 갈 게시물을 표시하고 있었다.
  • 하지만 사용자는 항상 페이지를 보고있는 것은 아니기 때문에
  • 몇 시간 후에는 더 새로운 기사가 수북히 쌓여
  • 결국 읽지 않은 기사는 영원히 접할 기회가 없어져 버린다.

해결

  • 그래서 Story Bumping 알고리즘이 등장했다.
  • Facebook은 단순히 최종 사용자가 확인한 후에 게시 된 기사만을 평가하는 것이 아니라,
  • 최근 읽지 않은 기사도 평가 대상에 추가했다.
  • 그래서 좀더 관심있는 기사를 볼 수 있게 되었다. (조금 오래되었더라도.)

평가

  • Facebook은 이미 웹 버전에서는 Story Bumping을 운용 중
  • 테스트 결과 Story Bumping은 좋아요, 공유 수를 5% 증가시켰다.
  • 또한 기업이나 개인 페이지에 대한 반응을 8% 증가시켰다.
  • 또한 “사용자가 관심을 가질 가능성이있는 기사” 를 57%에서 70% 로 크게 증가시켰다.

700 million people use the Facebook News Feed every day but many don’t understand how it decides what appears, so Facebook announced today it will start on how the feed algorithm is changing similar to how Google does. So Facebook will soon start to show these rapid real-time updates in chronological order so you see the most recent update first and the oldest one last. Facebook’s first post will be about “Story Bumping”, which pushes stories you haven’t seen above ones you have.
